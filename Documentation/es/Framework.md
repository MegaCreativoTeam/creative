## Estructura de directorios

```
├───api
│   ├───kernel
│   ├───logs
│   └───versions
│       └───v1
│           ├───controllers
│           ├───models
│           └───views
├───application

│   ├───framework
│   │   ├───console
│   │   │   ├───commands
│   │   │   └───templates
│   │   ├───kernel
│   │   │   ├───components
│   │   │   │   └───templates
│   │   │   ├───interfaces
│   │   │   ├───sessions
│   │   ├───libs
│   │   └───tpl
│   ├───helpers (Funciones predefinidas)
│   ├───hooks (Disparadores)
│   ├───langs (Traducciones)
│   |───logs
│   |───middleware (Capa intermedia)
│   |───modules
│   |───mvc
│   |───temporal
```
